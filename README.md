# AetherLink

![AetherLink Banner](aetherlink-pc\src-tauri\icons\icon.png)

**AetherLink** – это приложение для удалённого управления компьютером (Windows) с телефона (Android) через локальную сеть.  
Просто, быстро и безопасно.  

> 🚀 Управляй ПК с дивана, не вставая с места.

## ✨ Возможности

- **Безопасный режим**  
  Выключение, сон, блокировка экрана, управление громкостью, управление медиа (плей/пауза/следующий трек) – всё одной кнопкой.

- **Автоматизация (профили)**  
  Создавайте на компьютере профили (запуск программ, скриптов, команд) и запускайте их с телефона в один клик.

- **Режим разработчика** *(опционально)*  
  Полный доступ к командной строке с телефона (PowerShell / CMD) для продвинутых пользователей.

- **QR-привязка**  
  Сканируйте QR-код на экране ПК – устройство автоматически свяжется и получит доступ.

- **Работа в фоне**  
  ПК-клиент сворачивается в трей и не мешает работе. Вы получаете полный контроль без лишних окон.

- **Автозагрузка**  
  ПК-сервер можно добавить в автозагрузку – всегда готов к приёму команд.

## 🚀 Быстрый старт

1. **Установите приложение** на компьютер и телефон.
2. **Запустите ПК-версию** – Откройте QR‑код.
3. **Откройте приложение на телефоне**, нажмите «Войти» и сканируйте QR‑код.
4. Готово! Теперь вы можете отправлять команды с телефона.

> ℹ️ Убедитесь, что телефон и компьютер находятся в одной локальной сети Wi‑Fi.

## 📝 Профили: что писать

| Тип | Пример | Что делает |
| :--- | :--- | :--- |
| **RunBat** | `start "MyGame" /high "C:\Games\game.exe"` | Запускает игру с высоким приоритетом (больше ресурсов ЦП) |
| | `start https://github.com/r1cew/AetherLink` | Открывает ссылку в браузере |
| **RunExe** | Программа: `C:\Steam\steam.exe`<br>Аргументы: `-applaunch 730` | Запускает **CS2** из Steam |
| **PowerShell** | `Start-Process "notepad" -WindowStyle Hidden` | Открывает блокнот в фоне |

> 💡 **Совет:** Команда `start` (только для RunBat) может открывать файлы, папки и даже веб-ссылки.

## 🛠️ Функции в деталях

| Режим | Доступные действия |
|-------|--------------------|
| **Безопасный** | Выключение, сон, блокировка, громкость, управление музыкой (плей/пауза/следующий/предыдущий). Запуск заранее созданных профилей (программы, бат‑файлы, PowerShell скрипты). Создание профилей – только на ПК (безопасность). | 
| **Разработчик** | Создание своих собственных команд с телефона (пользовательские скрипты). Режим включается вручную на ПК для каждого устройства. |

---

## 🔒 Безопасность

- Все данные шифруются (протокол Noise XX).
- При привязке проверяется публичный ключ сервера – защита от подмены (MITM).
- Одноразовый токен для QR действителен 2 минуты.
- Режим разработчика отключён по умолчанию – нужно включать вручную на ПК.
